<h1>Football Score Keeper App</h1>

===================

This is a project after Lesson 2A and 2B. It is an app for keeping the score of football matches.

First there is a progress bar:
<img src="/screenshots/Screenshot_1487599628.png" width="281" height="500">

After 3 seconds the progress bar will be hiden and you have to choose the names of the teams:

<img src="/screenshots/Screenshot_1487599631.png" width="281" height="500">  <img src="/screenshots/Screenshot_1487599643.png" width="281" height="500">

After you press the "Football buttton" the game will "start":

<img src="/uploads/default/original/4X/9/c/a/9ca5d97d90c543c4e0fd3b6d3dbbab01fb4322b0.png" width="281" height="500">

By pressing green buttons for goals, yellow or red card or corners, you will add a new value for the current team:

<img src="/uploads/default/original/4X/a/d/2/ad25aceec21bf5939ef88b397bd4302a03922fde.png" width="281" height="500">

After you decide that the game is finished, you can press "finish game" and depends on the result you will see if there is a winner or not:

<img src="/uploads/default/original/4X/9/5/8/958d13591cc2fc46cacb50719cd52e7731ecc808.png" width="281" height="500">   OR   <img src="/uploads/default/original/4X/1/2/d/12dc8394310174e0ba770bada3afeb5d90d54011.png" width="281" height="500">
